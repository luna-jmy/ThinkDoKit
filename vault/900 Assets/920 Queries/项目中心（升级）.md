>*有主项目文件的默认显示主项目文件名。无主项目文件的显示文件夹名。最多预览5个项目笔记，可在编辑模式下修改代码的第2行参数 `maxNotesDisplay = 20` ，调整预览笔记数量，目前设置的上限是20条。*

```dataviewjs
dv.view("projectOverview", {
    maxNotes: 20
});
```
