### 添加新笔记的方法
1. 通过快速开始页中的按钮新建[[👉从这里开始 Start from here!]]。新建的笔记会自动套用模板并储存在对应的文件夹，但[[TPL-ResearchNote]]除外，[[TPL-ResearchNote]]需要点击笔记内的按钮（如下图👇）选择存储位置。移动的分类的依据是笔记的元数据`source_type`
   ![[Pasted image 20250604205109.png]]
2. 通过键入空白双链新建笔记 `[[新笔记]]` ，新建的笔记默认存储在 `000 Inbox` （可在Obsidian设置中修改），无模板套用。
3. 快捷键 `ctrl + n` 建立新笔记，存储位置同上，无模板套用。

### 如何在空白的笔记中套用模板
1. 方法一：通过templater插件添加。在笔记**第一行开头位置**， `ctrl + p` 打开命令栏， 选择`templater: Open insert template model`，选择要套用的模板。记住要在第一行开头位置插入，否则可能无法正确识别笔记的YAML元数据。 
2. 方法二：通过obsdian自带模板指令添加。 `ctrl + p` 打开命令栏，输入 `模板：插入模板` 。此方法不会自动运行templater插件中的语句，可能出现原始代码，可以通过快捷键 `alt + r` 手动运行templater代码。推荐直接使用templater插件或QuickAdd插入模板。
注意：模板必须放在指定的模板文件夹才生效！目前设置的模板文件夹是 `900 Assets/910 Templates`，不建议修改模板存放位置，修改后所有与模板相关的插件设置都需要手动修改。

>Tips: 如何在空白笔记中套用模板？ ;; 可以通过 `ctrl + p` 打开命令栏选择使用templater插件插入模板或使用Obsidian自带的模板插入功能。

### 如何快速输出指令
- 在Obsidian设置→选项→核心插件 中找到斜杠命令并开启。目前已设置开启。
- 在==行首==或正文任意位置（需要空一格），然后输入 `/` 即可快速打开obsidian命令栏。 

>Tips：如何在笔记编辑状态快速唤醒Obsidian命令？ ;; 在行首或正文任意位置（空一格）输入 `/` ，也可以使用快捷键 `ctrl + p` 。

### 如何通过指令快速添加笔记
- 输入 `/` 或使用快捷键 `ctrl + p` 打开命令栏，输入QuickAdd中设置的快捷指令（见下图👇），点亮⚡标志的QuickAdd命令可以直接通过命令栏使用，未点亮的需要先运行QuickAdd后才能选择。
![[Pasted image 20250604205200.png]]

>Tips：如何快速添加模板笔记？ ;; 输入 `/` 快速唤醒或使用快捷键 `ctrl + p` 打开命令栏，然后输入QuickAdd中设置的快捷添加模板指令

### 如何快速清空笔记/模板里的提示？
- 模板里设置了很多填写提示，一个一个删除比较麻烦，所以我在QuickAdd里预设了一个清空提示的快捷键。打开命令栏输入clear，找到👇这个命令，执行就可以清空笔记中的提示啦。![[Pasted image 20250604205226.png]]
- 注意：这个功能只能清空 ***引用 + 斜体*** 格式的提示。也就是markdown为 `>*提示*` 格式的提示词。
- 如果希望永久清空提示，可以在 `900 Assets/910 Templates` 文件夹下找到对应的笔记，删除提示即可。

>Tips：如何快速清空笔记里的提示？ ;; 使用快捷键 `alt+c` 或在命令栏里找到ClearReminder这个命令。

### 如何显示/隐藏笔记属性
笔记的属性数据有时会比较长，影响笔记查看时的体验，可以通过以下几种方法隐藏属性信息。
1. 在Obsidian设置 → 编辑器 → 笔记属性 选择隐藏即可。但建议同时在功能栏开启属性查看。笔记属性对于笔记管理非常重要，一定要维护。 ![[Pasted image 20250604205245.png]]
2. 通过自定义的CSS代码片段实现笔记属性隐藏。目前我设计了2种模式，一种是鼠标悬停的时候显示，移开时隐藏（目前的默认设置）。另一种是默认显示，但是可以点击收起。可以根据需要选择你喜欢的方式。
   注意：Obsidian编辑器里要选择显示笔记属性再开启CSS。另外，只能开启一个，不要两个代码同时开启。 ![[Pasted image 20250604205300.png]]

>Tips：如何取消自动隐藏笔记属性功能？ ;; 在Obsidian设置 → 外观 最下方找到CSS代码片段，关闭自动隐藏的代码片段即可。

### 如何关闭浮动目录条
- 浮动目录条是通过插件 `floating toc` 实现的，如果不希望页面有浮动目录显示，可以在插件设置里关闭或调整显示形式。

>Tips：如何关闭笔记上的浮动目录条？ ;; 在Obsidian设置 → 第三方插件里找到 `floating toc` 插件，关闭或进入设置页面调整显示方式。
