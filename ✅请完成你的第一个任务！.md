## 清理笔记中的示例文件

下方查询列出了笔记中的所有示例文件，请：
1. 点击表格链接进入对应示例文件，查看通过模板创建的范例笔记。
2. 点击笔记右上方“更多选项”→“删除文件”。删除后此列表会自动更新。
3. 全部删除后，就可以开始记录你自己的内容啦。
4. 如果已经对这个仓库熟悉了，可以点击以下按钮一键删除demo笔记。

```button
name 一键删除示例笔记
type command
action QuickAdd: DeleteDemo
color yellow
```

>*你也可以保留示例笔记作为功能测试用的文件，或者通过[[👉从这里开始 Start from here!|快速开始页]]新建笔记后，将笔记元数据 `type` 改为 `demo`，就可以在下表中查看到了。*

```dataview
TABLE WITHOUT ID
  file.link AS "笔记名称",
  type AS "类别",
  file.path AS "路径"
FROM ""
WHERE type = "demo"
WHERE !contains(file.path, "900 Assets")
SORT file.path ASC
```
